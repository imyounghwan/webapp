## 🎯 프로젝트 개요

**AutoAnalyzer**는 49개 공공기관의 국민평가 데이터를 **제이콥 닐슨(Jakob Nielsen)의 10가지 사용성 원칙**에 따라 분석하여 **구체적인 UI/UX 진단과 개선 방안**을 자동으로 생성하는 웹 기반 분석 시스템입니다.

### 핵심 성과
- ✅ **Nielsen 10원칙 → 25개 세부 항목** 확장 분석
- ✅ **49개 기관 × 6개 연령대 = 294 데이터포인트** 처리
- ✅ **각 항목별 구체적 진단 + 개선 방안** 자동 생성
- ✅ **인터랙티브 웹 UI** (FTP 배포 가능)
- ✅ **687KB Nielsen 상세 분석 JSON** 데이터베이스

## 🔬 분석 예시: 고용노동부 (Nielsen 종합 4.48/5.0)

### 강점 분석 ✅

**N3.1 뒤로가기/취소 기능: 4.75/5.0** (Excellent)
- **진단**: 모든 페이지에 '이전' 버튼이 명확히 표시되며, 브라우저 뒤로가기도 정상 작동합니다.
- **평가**: 우수한 사용자 제어 기능 구현

**N5.3 중요 작업 확인: 4.58/5.0** (Excellent)
- **진단**: 삭제, 제출 등 중요한 작업 전 '정말 삭제하시겠습니까?' 확인 메시지가 나타납니다.
- **평가**: 효과적인 오류 예방 메커니즘

### 개선 기회 ⚠️

**N3.2 임시 저장 기능: 3.40/5.0** (Poor)
- **진단**: 임시 저장 기능이 없어 사용자가 실수로 페이지를 닫으면 모든 입력이 날아갑니다.
- **개선 방안**: 
  - 5분마다 자동 저장 구현
  - '마지막 저장: 2분 전' 상태 메시지 표시
  - '임시 저장' 버튼을 우측 하단에 배치
- **예상 효과**: 3.40 → 4.50 (+1.10점)

**N7.1 단축키 제공: 2.97/5.0** (Poor)
- **진단**: 키보드만으로는 작업이 불가능하며, 마우스에 의존해야 합니다.
- **개선 방안**:
  - 주요 버튼에 'Ctrl+S로 저장' 힌트 표시
  - 도움말에 단축키 목록 제공
  - Tab/Enter/ESC 키 네비게이션 지원
- **예상 효과**: 2.97 → 4.20 (+1.23점)

### 종합 개선 시나리오
- 현재 Nielsen 종합: **4.48/5.0**
- 주요 5개 항목 개선 시: **4.48 → 4.72 (+0.24점)**
- **종합 평가 등급**: S (4.5+) 유지 + 약점 보완

## 💻 기술 구현

### 분석 엔진 (Python)
- **nielsen_analyzer.py**: Nielsen 10원칙 기반 25개 항목 분석
  - Q1~Q10 데이터를 Nielsen 항목으로 매핑
  - 점수 구간별 자동 진단 (Excellent/Good/Poor)
  - 항목별 구체적 개선 방안 생성
- **age_analyzer.py**: 연령대별 분석
- **detailed_report_generator.py**: 상세 리포트 생성

### 웹 인터페이스 (HTML/CSS/JS)
- **Nielsen 상세 리포트 모달**:
  - 10개 원칙별 점수 시각화
  - 25개 항목별 진단 및 개선 방안 표시
  - 색상 코딩 (Green/Cyan/Red)
  - 개선 우선순위 자동 정렬
- **인터랙티브 차트**: Chart.js 활용
- **반응형 디자인**: 모바일/태블릿 지원

### 데이터 구조
- **nielsen_detailed_reports.json**: 687KB
  - 49개 기관 × 25개 항목 = 1,225개 진단
  - 각 진단당 평균 560자의 상세 설명
