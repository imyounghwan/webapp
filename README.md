## 시스템 한계 및 권장사항

### ⚠️ HTML 자동 분석의 한계
1. **동적 콘텐츠 미감지**: JavaScript로 생성되는 콘텐츠는 분석 불가
2. **주관적 항목 평가 어려움**: "사용하기 편한가", "디자인이 좋은가" 등
3. **실제 기능 테스트 불가**: 로그인, 신청 등 실제 작동 여부는 수동 검증 필요

### 💡 권장 사용법
- **1차 스크리닝 도구**: 자동 분석 60~70% + 전문가 검증 30~40%
- **대량 사이트 사전 평가**: 100개 사이트를 자동으로 평가 후 우선순위 선정
- **정기 모니터링**: 주기적 자동 평가로 품질 변화 추적
- **최종 보고서**: 반드시 전문가가 직접 검증 후 발행
