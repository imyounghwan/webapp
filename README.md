## 주요 기능

### 자동 평가
- **KRDS UI/UX 43개 항목** 자동 평가
- HTML 구조 분석 기반 객관적 평가
- 준수율 기반 점수 산정 (0-100점)

### 사이트별 최적화
- 10개 기관별 보정 계수 적용
- HTML 품질 vs 실제 사용성 차이 보정
- 과대평가/과소평가 문제 해결

### 지원 기능
- JavaScript 리다이렉트 자동 추적 (최대 3회)
- 무한 루프 방지
- Cookie 기반 인증 지원

## 기술 스택

- **Backend**: Hono Framework
- **Frontend**: HTML + TailwindCSS + Vanilla JS
- **Database**: Cloudflare D1 (SQLite)
- **Deployment**: Cloudflare Pages
